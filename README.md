# LUT Visualizer

3D LUT（`.cube`ファイル）の色変換を可視化するブラウザツール。外部ライブラリ・通信なしの単一 HTML です。

## 使い方
1. `index.html` を開く
2. 「.cubeファイルを開く」または本体へのドラッグ&ドロップでLUTを読み込む（手元にファイルが無ければ「サンプルLUTを試す」）
3. 3パネルで確認:
   - **3Dプロット**: LUTの各グリッド点を入力RGB位置に配置し、出力色で着色。ドラッグで回転。白いワイヤーフレームは無加工の基準立方体
   - **Before/After**: 色相グラデーションのテスト画像にLUTを適用した結果を並べて比較
   - **RGB応答カーブ**: 対角線（R=G=B）上の入出力関係をチャンネルごとに表示

## 対応形式
- Adobeの`.cube`形式（3D LUT）のみ。`TITLE` / `LUT_3D_SIZE` / `DOMAIN_MIN` / `DOMAIN_MAX` に対応、1D LUTは非対応。
- 出力色サンプリングはトライリニア補間。

## 技術メモ
- 3D点群は自前の透視投影（Three.js等は不使用）。LUTサイズが大きい場合は表示点数を間引いて描画負荷を抑制。
- 色は原則データそのもの（LUTの出力RGB）。UIのアクセント・RGBカーブの3色のみ配色ルールに沿って選定・検証済み。
